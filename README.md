## 安装

注意：需使用 nodejs 长期维护版本，如：[18.x、20.x]，能保证项目的稳定运行。

```bash

# 进入项目
cd train-admin

# 安装依赖
npm install

# 运行项目
npm run dev

# 打包发布
npm run build
```
